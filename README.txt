Instructions:
1. Power off the computer on Minecraft
2. Go to VirtualBox
3. Select the MCVmComputersVm and open its settings
4. Go to network>Advanced
5. Switch the adapter type to Paravirtualized Network (virtio-net)
6. Press OK
7. Attach this cd and power on the vm again
8. Go to the cd drive and open Install.cmd
9. Select the OS
10. Confirm all warnings to install the driver
11. Done!
